# In 和 NotIn 函数性能优化报告

## 优化策略

**V11 组合优化方案**：
- 预先将所有值转换为 `reflect.Value`（避免重复反射）
- 检测类型统一性
- 对于统一类型使用 map 优化（O(1)查找）
- 对于 int/string/float 类型专用快速路径
- 混合类型降级为线性查找

## 关键优化点

1. **预转换反射值**：避免每次迭代都调用 `reflect.ValueOf()`
2. **类型专用快速路径**：int/string/float 使用 map 查找（O(1)）
3. **统一类型检测**：同类型值使用 map，混合类型线性查找
4. **零内存分配**：所有优化方案都是 0 B/op

**V11组合优化方案在所有场景下都表现优异**：
