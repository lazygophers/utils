# xtime955

Time constants for the 955 work schedule (9am-5pm, 5 days a week). Extends the standard `time` package with work-day, work-week, and work-month constants.

```go
import "github.com/lazygophers/utils/xtime/xtime955"

// Work day = 8 hours
xtime955.WorkDay

// Work week = 5 work days
xtime955.WorkWeek
```

For the full list of exported constants, see [pkg.go.dev](https://pkg.go.dev/github.com/lazygophers/utils/xtime/xtime955).

See also [xtime](/en/modules/time/xtime).
