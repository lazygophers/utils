# xtime007

Time constants for the 24/7 work schedule (0am-12am, 7 days a week). Extends the standard `time` package with work-day, work-week, and work-month constants.

```go
import "github.com/lazygophers/utils/xtime/xtime007"

// Work day = 24 hours
xtime007.WorkDay

// Work week = 7 days
xtime007.WorkWeek
```

For the full list of exported constants, see [pkg.go.dev](https://pkg.go.dev/github.com/lazygophers/utils/xtime/xtime007).

See also [xtime](/en/modules/time/xtime).
