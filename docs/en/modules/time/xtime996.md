# xtime996

Time constants for the 996 work schedule (9am-9pm, 6 days a week). Extends the standard `time` package with work-day, work-week, and work-month constants.

```go
import "github.com/lazygophers/utils/xtime/xtime996"

// Work day = 12 hours
xtime996.WorkDay

// Work week = 6 work days
xtime996.WorkWeek
```

For the full list of exported constants, see [pkg.go.dev](https://pkg.go.dev/github.com/lazygophers/utils/xtime/xtime996).

See also [xtime](/en/modules/time/xtime).
