# i18n

Validator supports error messages in 12 languages.

## Supported Languages

English (`en`) and Chinese (`zh`) are always available. Others require build tags:

| Language | Code | Status |
|----------|------|--------|
| English | `en` | Always enabled |
| Simplified Chinese | `zh` | Always enabled |
| Traditional Chinese | `zh-TW` | Build tag `lang_zh_tw` |
| Japanese | `ja` | Build tag `lang_ja` |
| Korean | `ko` | Build tag `lang_ko` |
| Arabic | `ar` | Build tag `lang_ar` |
| German | `de` | Build tag `lang_de` |
| Spanish | `es` | Build tag `lang_es` |
| French | `fr` | Build tag `lang_fr` |
| Italian | `it` | Build tag `lang_it` |
| Portuguese | `pt` | Build tag `lang_pt` |
| Russian | `ru` | Build tag `lang_ru` |

## Per-Request Language Switch

```go
func HandleRequest(user *User, lang string) error {
    validator.SetLocale(lang)
    return validator.Struct(user)
}
```

## Custom Translations

```go
validator.RegisterTranslation("en", "myrule", "{field} failed custom rule")
validator.RegisterTranslation("zh", "myrule", "{field} 不符合自定义规则")
```
