# 多語言

Validator 支援 12 種語言的錯誤訊息。

## 支援的語言

英文（`en`）和中文（`zh`）始終可用，其他語言需啟用：

| 語言 | 代碼 | 狀態 |
|------|------|------|
| 英文 | `en` | 始終啟用 |
| 簡體中文 | `zh` | 始終啟用 |
| 繁體中文 | `zh-TW` | Build tag `lang_zh_tw` |
| 日語 | `ja` | Build tag `lang_ja` |
| 韓語 | `ko` | Build tag `lang_ko` |
| 阿拉伯語 | `ar` | Build tag `lang_ar` |
| 德語 | `de` | Build tag `lang_de` |
| 西班牙語 | `es` | Build tag `lang_es` |
| 法語 | `fr` | Build tag `lang_fr` |
| 義大利語 | `it` | Build tag `lang_it` |
| 葡萄牙語 | `pt` | Build tag `lang_pt` |
| 俄語 | `ru` | Build tag `lang_ru` |

## 按請求切換語言

```go
func HandleRequest(user *User, lang string) error {
    validator.SetLocale(lang)
    return validator.Struct(user)
}
```

## 自訂翻譯

```go
validator.RegisterTranslation("zh-TW", "myrule", "{field} 不符合自訂規則")
validator.RegisterTranslation("en", "myrule", "{field} failed custom rule")
```
