# 模組概覽

LazyGophers Utils 提供了 20+ 專業模組，涵蓋 Go 開發的各個方面。

## 🔧 核心工具

### must.go

錯誤斷言工具，簡化錯誤處理流程。

**主要功能：**
- `Must()` - 斷言操作成功，失敗時 panic
- `MustSuccess()` - 斷言錯誤為 nil
- `MustOk()` - 斷言第二個返回值為 true

**示例：**
```go
data := utils.Must(loadData())
utils.MustSuccess(config.Load(&cfg, "config.json"))
value := utils.MustOk(getValue())
```

### orm.go

資料庫操作工具，提供便捷的資料轉換方法。

**主要功能：**
- `Scan()` - 將資料庫結果掃描到結構體
- `Value()` - 將結構體轉換為資料庫值

### validator

資料驗證模組，支援結構體驗證。

**主要功能：**
- `Validate()` - 驗證結構體資料
- 支援內建驗證規則：`required`, `email`, `min`, `max` 等

## 🍭 資料處理

### candy

類型轉換工具，提供零分配的類型轉換。

**主要功能：**
- `ToInt()` - 字串轉整數
- `ToFloat()` - 字串轉浮點數
- `ToBool()` - 字串轉布林值
- `ToString()` - 任意類型轉字串
- `ToSlice()` - 任意類型轉切片
- `ToMap()` - 任意類型轉 map

**性能：** 零分配，比標準庫快 3.2 倍

### json

增強型 JSON 處理，提供更好的錯誤訊息。

**主要功能：**
- `Marshal()` - JSON 編碼
- `Unmarshal()` - JSON 解碼
- 更友好的錯誤訊息

### stringx

字串工具，支援 Unicode 感知操作。

**主要功能：**
- `Rand()` - 生成隨機字串
- `Reverse()` - 反轉字串
- `Truncate()` - 截斷字串
- Unicode 感知的字串操作

### anyx

interface{} 助手，提供類型安全的 any 操作。

**主要功能：**
- 類型安全的 any 操作
- 泛型支援

## ⏰ 時間與調度

### xtime

高級時間處理，支援農曆、生肖、節氣等。

**主要功能：**
- `NowCalendar()` - 獲取當前日曆
- `LunarDate()` - 獲取農曆日期
- `Animal()` - 獲取生肖
- `CurrentSolarTerm()` - 獲取當前節氣
- `Format()` - 格式化日期

**特殊功能：**
- 🌙 農曆支援
- 🐲 生肖計算
- 🌾 節氣計算

### xtime996

996 工作時間計算。

**主要功能：**
- `IsWorkTime()` - 判斷是否為工作時間
- 工作時間計算

### xtime955

955 工作時間計算。

**主要功能：**
- `IsWorkTime()` - 判斷是否為工作時間
- 平衡工作時間支援

### xtime007

24/7 運營時間工具。

**主要功能：**
- 全天候時間工具
- 始終在線的時間處理
