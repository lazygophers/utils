# xtime007

全天候工作制時間常量（0:00-24:00，每週 7 天）。基於標準庫 `time` 包擴展，提供工作日、工作週、工作月等常量。

```go
import "github.com/lazygophers/utils/xtime/xtime007"

// 工作日 = 24 小時
xtime007.WorkDay

// 工作週 = 7 天
xtime007.WorkWeek
```

完整的導出常量參見 [pkg.go.dev](https://pkg.go.dev/github.com/lazygophers/utils/xtime/xtime007)。
