# xtime996

996 工作制時間常量（9:00-21:00，每週 6 天）。基於標準庫 `time` 包擴展，提供工作日、工作週、工作月等常量。

```go
import "github.com/lazygophers/utils/xtime/xtime996"

// 工作日 = 12 小時
xtime996.WorkDay

// 工作週 = 6 個工作日
xtime996.WorkWeek
```

完整的導出常量參見 [pkg.go.dev](https://pkg.go.dev/github.com/lazygophers/utils/xtime/xtime996)。
