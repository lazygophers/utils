# 數據處理

提供類型轉換、JSON 處理、字符串操作等數據處理工具。

## 模組列表

- [candy](./candy.md) - 類型轉換糖
- [json](./json.md) - 增強的 JSON 處理
- [stringx](./stringx.md) - 字符串工具
- [anyx](./anyx.md) - Interface{} 輔助
- [bufiox](./bufiox.md) - 自訂数據掃描函數
- [country](./country.md) - ISO 3166-1 國家/地區數據 + 多語言名 + 時區/區號/官方語言/首都
- [currency](./currency.md) - ISO 4217 貨幣數據 + 多語言名（154 種）
