# 多语言

Validator 支持 12 种语言的错误消息。

## 支持的语言

英文（`en`）和中文（`zh`）始终可用，其他语言需启用：

| 语言 | 代码 | 状态 |
|------|------|------|
| 英文 | `en` | 始终启用 |
| 简体中文 | `zh` | 始终启用 |
| 繁体中文 | `zh-TW` | Build tag `lang_zh_tw` |
| 日语 | `ja` | Build tag `lang_ja` |
| 韩语 | `ko` | Build tag `lang_ko` |
| 阿拉伯语 | `ar` | Build tag `lang_ar` |
| 德语 | `de` | Build tag `lang_de` |
| 西班牙语 | `es` | Build tag `lang_es` |
| 法语 | `fr` | Build tag `lang_fr` |
| 意大利语 | `it` | Build tag `lang_it` |
| 葡萄牙语 | `pt` | Build tag `lang_pt` |
| 俄语 | `ru` | Build tag `lang_ru` |

## 按请求切换语言

```go
func HandleRequest(user *User, lang string) error {
    validator.SetLocale(lang)
    return validator.Struct(user)
}
```

## 自定义翻译

```go
validator.RegisterTranslation("zh", "myrule", "{field} 不符合自定义规则")
validator.RegisterTranslation("en", "myrule", "{field} failed custom rule")
```
