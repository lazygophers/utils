    # pgp

    面向 OpenPGP 场景的签名、加密与密钥处理能力。

    ## 适用场景

    - 需要和 PGP 兼容的消息或文件流转。
- 需要在工程中补齐签名/验签流程。

    ## 你会接触到什么

    - 适合放在跨系统安全交换或离线文件处理场景。

    ## 使用建议

    - 如果你的需求只是普通哈希或对称加密，优先看 `cryptox`。
