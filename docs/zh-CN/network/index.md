    # 网络与安全

    适合处理网络辅助、密码学能力、PGP 流程以及 URL 规范化。

    ## 这一组包含什么

    | 模块 | 定位 |
    | --- | --- |
    | `network` | 网络环境、地址与请求相关辅助 |
| `cryptox` | 哈希、加解密、密钥与标识符辅助 |
| `pgp` | OpenPGP 场景下的签名与加解密 |
| `urlx` | URL / query 规范化处理 |

    ## 阅读建议

    - cryptox 是能力面最广的安全包，应按具体算法进入。
- urlx 很轻，适合做规范化拼接中的最后一步。
