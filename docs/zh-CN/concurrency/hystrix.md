    # hystrix

    面向依赖保护的熔断与降级能力。

    ## 适用场景

    - 调用外部系统时，希望在故障放大前快速失败。
- 需要把超时、错误率和回退策略统一管理。

    ## 你会接触到什么

    - 适合保护数据库、RPC、HTTP、消息投递等外部依赖。

    ## 使用建议

    - 把它放在依赖边界层最合适，不要让业务层直接感知过多熔断细节。
