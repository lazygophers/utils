    # 并发与控制流

    适合组织任务执行、等待条件、熔断保护、事件流与去重执行。

    ## 这一组包含什么

    | 模块 | 定位 |
    | --- | --- |
    | `routine` | goroutine / worker pool 组织 |
| `wait` | 等待、重试、超时与轮询控制 |
| `hystrix` | 熔断与降级保护 |
| `singledo` | 同一工作只执行一次 |
| `event` | 发布订阅式事件分发 |

    ## 阅读建议

    - 如果你在解决并发访问同一资源的问题，先看 singledo。
- 如果你在做外部依赖保护，先看 hystrix。
