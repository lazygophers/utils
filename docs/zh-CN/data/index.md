    # 数据处理

    适合处理转换、字符串规整、集合操作以及 JSON 编解码。

    ## 这一组包含什么

    | 模块 | 定位 |
    | --- | --- |
    | `candy` | 转换、集合、映射与切片辅助的重型工具包 |
| `json` | 围绕 JSON 编解码与文件读写的工程化封装 |
| `stringx` | 命名风格转换与字符串辅助 |
| `anyx` | 围绕 any / interface 的通用处理辅助 |

    ## 阅读建议

    - candy 功能面很广，适合先按问题类别读。
- json 更偏工程封装，不只是 Marshal/Unmarshal。
- stringx 适合做命名转换与展示层处理。
