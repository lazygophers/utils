    # anyx

    围绕 `any` / `interface{}` 值做规整、抽取和桥接的辅助包。

    ## 适用场景

    - 你在边界层接收动态数据。
- 你需要把不稳定输入统一成更容易处理的形式。

    ## 你会接触到什么

    - 适合与 `candy`、`json`、`validator` 组合使用，把动态输入逐步收敛成稳定结构。

    ## 使用建议

    - 如果业务模型已经稳定，优先直接建结构体；`anyx` 更适合边界和过渡层。
