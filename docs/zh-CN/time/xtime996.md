    # xtime996

    在固定的 996 制度假设下解释工作时段与相关时间逻辑。

    ## 适用场景

    - 你要围绕固定制度做工作时间判断。
- 你需要让“是否工作时段”有明确制度语义。

    ## 你会接触到什么

    - 它不是对 `xtime` 的别名，而是独立的制度假设包。

    ## 使用建议

    - 只有当你的业务确实使用这类制度假设时再引入；否则先停留在 `xtime`。
