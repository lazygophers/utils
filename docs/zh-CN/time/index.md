    # 时间与调度

    适合需要日历、农历、节气或固定工作制度计算的场景。

    ## 这一组包含什么

    | 模块 | 定位 |
    | --- | --- |
    | `xtime` | 日历、农历、节气与时间常量 |
| `xtime996` | 按 996 制度解释工作时段 |
| `xtime955` | 按 955 制度解释工作时段 |
| `xtime007` | 按全天候制度解释工作时段 |

    ## 阅读建议

    - xtime 不只是格式化时间，更接近领域模型。
- 007 / 955 / 996 子包表示不同制度假设，不是别名。
