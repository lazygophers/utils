# 模块概览

LazyGophers Utils 提供了 20+ 专业模块，涵盖 Go 开发的各个方面。

## 🔧 核心工具

### must.go

错误断言工具，简化错误处理流程。

**主要功能：**
- `Must()` - 断言操作成功，失败时 panic
- `MustSuccess()` - 断言错误为 nil
- `MustOk()` - 断言第二个返回值为 true

**示例：**
```go
data := utils.Must(loadData())
utils.MustSuccess(config.Load(&cfg, "config.json"))
value := utils.MustOk(getValue())
```

### orm.go

数据库操作工具，提供便捷的数据转换方法。

**主要功能：**
- `Scan()` - 将数据库结果扫描到结构体
- `Value()` - 将结构体转换为数据库值

### validator

数据验证模块，支持结构体验证。

**主要功能：**
- `Validate()` - 验证结构体数据
- 支持内置验证规则：`required`, `email`, `min`, `max` 等

## 🍭 数据处理

### candy

类型转换工具，提供零分配的类型转换。

**主要功能：**
- `ToInt()` - 字符串转整数
- `ToFloat()` - 字符串转浮点数
- `ToBool()` - 字符串转布尔值
- `ToString()` - 任意类型转字符串
- `ToSlice()` - 任意类型转切片
- `ToMap()` - 任意类型转 map

**性能：** 零分配，比标准库快 3.2 倍

### json

增强型 JSON 处理，提供更好的错误消息。

**主要功能：**
- `Marshal()` - JSON 编码
- `Unmarshal()` - JSON 解码
- 更友好的错误消息

### stringx

字符串工具，支持 Unicode 感知操作。

**主要功能：**
- `Rand()` - 生成随机字符串
- `Reverse()` - 反转字符串
- `Truncate()` - 截断字符串
- Unicode 感知的字符串操作

### anyx

interface{} 助手，提供类型安全的 any 操作。

**主要功能：**
- 类型安全的 any 操作
- 泛型支持

## ⏰ 时间与调度

### xtime

高级时间处理，支持农历、生肖、节气等。

**主要功能：**
- `NowCalendar()` - 获取当前日历
- `LunarDate()` - 获取农历日期
- `Animal()` - 获取生肖
- `CurrentSolarTerm()` - 获取当前节气
- `Format()` - 格式化日期

**特殊功能：**
- 🌙 农历支持
- 🐲 生肖计算
- 🌾 节气计算

### xtime996

996 工作时间计算。

**主要功能：**
- `IsWorkTime()` - 判断是否为工作时间
- 工作时间计算

### xtime955

955 工作时间计算。

**主要功能：**
- `IsWorkTime()` - 判断是否为工作时间
- 平衡工作时间支持

### xtime007

24/7 运营时间工具。

**主要功能：**
- 全天候时间工具
- 始终在线的时间处理
