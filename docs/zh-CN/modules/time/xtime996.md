# xtime996

`xtime996` 面向的是**固定工作制语义**下的时间判断与计算。它适合“规则先于代码”的场景：先确认制度模型，再决定是否接入。

## 适合什么场景

- 需要基于 996 规则计算工作时段或判断是否处于工作时间。
- 想把排班 / 工时规则从业务代码里抽离出来。
- 希望和 `xtime` 的日历能力一起使用。

## 使用建议

- 这里表达的是特定制度模型，不应默认套用到所有团队或地区。
- 业务里如果存在节假日、调休或特殊班次，仍需要额外规则层补充。
- 如果只是要一般性的日历、农历和节气信息，请先看 [xtime](/modules/time/xtime)。
