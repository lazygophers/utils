# xtime955

`xtime955` 用于表达**955 工作制**下的时间语义，适合把“是否上班、是否休息、当前所处制度时间段”这类判断收口到单独模块。

## 适合什么场景

- 需要围绕 955 规则做状态判断或时间计算。
- 想把固定制度的工时规则抽离出业务代码。
- 需要与普通日历信息一起使用，但不想手写制度判断。

## 使用建议

- 这是特定制度模型，不等同于所有“正常工作日”概念。
- 如果业务需要精细节假日或地区差异，仍要在上层补充规则。
- 对制度边界存在争议时，先和产品/业务确认语义，再接入代码。
