# randx

`randx` 提供**随机值与随机辅助**。当你需要随机字符串、数值、采样选择或测试辅助随机行为时，可以先看这里。

## 适合什么场景

- 生成随机 ID、随机字符串或测试输入。
- 需要轻量随机辅助而不想重复写样板。
- 与假数据、默认值或本地调试流程搭配使用。

## 使用建议

- 随机工具要先区分“测试辅助”与“安全随机”，不要把普通随机直接用于安全敏感场景。
- 如果目标是构造更接近真实业务的数据，而不只是随机字符，请看 [fake](/modules/dev/fake)。
- 涉及可重现测试时，要明确随机种子策略。
