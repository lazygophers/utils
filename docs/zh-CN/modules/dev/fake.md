# fake

`fake` 主要用于**生成测试、演示或初始化所需的假数据**。它的重点不是随机性本身，而是给开发、联调和样例提供可用数据源。

## 适合什么场景

- 需要批量生成示例用户、地址、名称等测试数据。
- 文档、演示环境或本地开发需要更接近真实的数据形态。
- 想减少手写样例数据的重复劳动。

## 使用建议

- 假数据适合开发和测试，不要混入生产真实链路。
- 如果你需要的是纯随机数或随机字符串，请继续看 [randx](/modules/dev/randx)。
- 该包可能包含较多 locale / embed 数据，调整行为前先确认数据来源和加载方式。
