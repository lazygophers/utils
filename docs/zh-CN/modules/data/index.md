# 数据处理

提供类型转换、JSON 处理、字符串操作等数据处理工具。

## 模块列表

- [candy](./candy.md) - 类型转换糖
- [json](./json.md) - 增强的 JSON 处理
- [stringx](./stringx.md) - 字符串工具
- [anyx](./anyx.md) - Interface{} 辅助
- [bufiox](./bufiox.md) - 数据扫描函数
- [country](./country.md) - ISO 3166-1 国家/地区数据 + 多语言名 + 时区/区号/官方语言/首都
- [currency](./currency.md) - ISO 4217 货币数据 + 多语言名（154 种）
