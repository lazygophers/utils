# candy

`candy` 是一组偏工程实用的**类型转换、集合处理与便捷辅助函数**。如果你经常在字符串、数字、布尔值、切片、映射之间做转换，它通常会比手写样板更直接。

## 适合什么场景

- 从配置、环境变量、表单或弱类型输入中取值并转换。
- 做切片、映射、去重、拼接、过滤之类的常见集合操作。
- 想让“转换失败时给默认零值”这种策略保持一致。

## 你会在这里关注什么

- 各类 `ToXxx` 转换函数。
- 面向切片 / 映射的便捷操作。
- 对弱类型输入的容忍度以及默认返回策略。

## 使用建议

- `candy` 更适合做“输入整形”和“轻量转换”，不是严格的数据校验工具。
- 遇到关键字段时，转换成功并不代表合法，仍建议再配合 [validator](/modules/core/validator)。
- 新版中文文档不再保留未经证明的 benchmark 数字；是否足够快，应以你的真实负载测试为准。
