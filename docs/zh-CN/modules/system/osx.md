# osx

`osx` 主要提供**文件、目录和操作系统路径相关的实用辅助**。如果你在写安装器、初始化脚本或本地资源管理逻辑，这个包值得先看。

## 适合什么场景

- 创建、复制、移动、检查文件与目录。
- 组织程序运行所需的本地资源路径。
- 想把常见的文件系统样板操作收口到统一位置。

## 使用建议

- 文件系统操作天然带副作用，调用方要对覆盖、权限、存在性检查保持明确预期。
- 平台相关路径行为要在目标系统上实测，不要只凭单一环境验证。
- 如果你只是要拿运行时目录信息，优先看 [runtime](/modules/system/runtime)。
