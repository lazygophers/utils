    # 核心工具

    适合放在项目最靠近启动与边界的位置：快速失败、数据库字段映射、结构体验证。

    ## 这一组包含什么

    | 模块 | 定位 |
    | --- | --- |
    | `must` | 初始化阶段的 fail-fast 辅助 |
| `orm` | 数据库字段与结构体之间的 JSON 映射桥 |
| `validator` | 结构体验证与本地化错误信息 |

    ## 阅读建议

    - 如果你想减少样板错误处理，先看 must。
- 如果数据库字段存的是 JSON 或字符串对象，先看 orm。
- 如果需要在配置或请求边界做规则校验，先看 validator。
