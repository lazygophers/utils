    # 系统与配置

    适合处理配置、路径、应用元信息、退出清理与系统交互。

    ## 这一组包含什么

    | 模块 | 定位 |
    | --- | --- |
    | `config` | 多格式配置加载与环境覆盖 |
| `runtime` | 运行时目录、栈与上下文辅助 |
| `osx` | 文件与操作系统交互辅助 |
| `app` | 应用级元信息与初始化支撑 |
| `atexit` | 退出回调与平台差异 |

    ## 阅读建议

    - config 往往和 validator 一起使用。
- atexit 的平台差异不能忽略。
- runtime / osx 更适合支撑基础设施层。
