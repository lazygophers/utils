    # runtime

    围绕运行时目录、环境、栈与执行上下文提供辅助能力。

    ## 适用场景

    - 想获取当前工作目录、可执行文件目录或用户目录。
- 想在基础设施层记录栈或诊断运行环境。

    ## 你会接触到什么

    - 适合作为 `config`、`osx`、`app` 等基础设施包的支撑层。

    ## 使用建议

    - 更偏平台/运行时边界，不建议直接把它们散落在业务代码里。
