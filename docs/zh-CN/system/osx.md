    # osx

    围绕文件、路径和操作系统交互的辅助包。

    ## 适用场景

    - 做文件存在性判断、路径定位或系统资源操作。
- 为 `config`、`runtime` 这类包提供底层支撑。

    ## 你会接触到什么

    - 文档更关注它在工程中的职责，而不是把所有文件辅助函数逐一列出来。

    ## 使用建议

    - 如果你只是做一般文件读写，仍然可以直接使用标准库；需要统一工程语义时再引入 osx。
