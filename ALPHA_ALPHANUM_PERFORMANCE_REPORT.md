## 实现方案

### 最优方案选择

基于性能测试结果，选择了以下方案：

1. **Alpha**: 单次遍历优化 + 位运算
   - 使用 `c | 0x20` 转换为小写
   - 简单范围检查 `'a'` 到 `'z'`
   - 代码简洁且性能优异

2. **Alphanum**: 单次遍历优化 + 位运算
   - 字母部分使用位运算转换为小写
   - 数字部分直接范围检查
   - 平衡了性能和可读性

### 实现代码

#### Alpha 优化实现

```go
func Alpha() ValidatorFunc {
	return func(fl FieldLevel) bool {
		s := fl.Field().String()
		if len(s) == 0 {
			return true
		}
		for i := 0; i < len(s); i++ {
			c := s[i] | 0x20 // 转换为小写
			if c < 'a' || c > 'z' {
				return false
			}
		}
		return true
	}
}
```

#### Alphanum 优化实现

```go
func Alphanum() ValidatorFunc {
	return func(fl FieldLevel) bool {
		s := fl.Field().String()
		if len(s) == 0 {
			return true
		}
		for i := 0; i < len(s); i++ {
			c := s[i]
			cl := c | 0x20 // 字母转换为小写
			if (cl < 'a' || cl > 'z') && (c < '0' || c > '9') {
				return false
			}
		}
		return true
	}
}
```
